# HTML_CALENDAR
```html``` ```css``` ```javascript``` 를 이용하여 년도와 첫달의 시작 지점을 입력으로 받아 입력받은 년도의 달력을 출력하는 
프로그램을 작성해보자.

## JavaScirpt Code

```c
int find_largest(int a[], int n)
{
  int i, max;

  max = a[0];
  for(i = 1; i < n; i++)
    if(a[i] > max)
      max = a[i];
  return max;
}
```
